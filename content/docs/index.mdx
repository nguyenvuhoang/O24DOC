## Quick Start

Or you can follow the steps below to clone the repository and run it locally.

<Steps>
    ### Clone this repository

    Using the GitHub CLI:
    ```bash
    gh repo clone nguyenvuhoang/O24DOC
    ```

    ### Install
    ```bash
    pnpm i
    ```

    ### Run the development server
    ```bash
    pnpm dev
    ```
</Steps>
