## Hướng dẫn nhanh

Hoặc bạn có thể làm theo các bước bên dưới để sao chép github và chạy nó trên máy tính cục bộ.

<Steps>
    ### Clone this repository

    Sử dụng giao diện dòng lệnh GitHub:
    ```bash
    gh repo clone nguyenvuhoang/O24DOC
    ```

    ### Install
    ```bash
    pnpm i
    ```

    ### Run the development server
    ```bash
    pnpm dev
    ```
</Steps>
